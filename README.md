# Hello World React App

A simple "Hello World" React application built with Vite.

## Prerequisites

Before running this app, you need to have the following installed:

- **Node.js** (version 16 or higher) - [Download here](https://nodejs.org/)
- **npm** (comes with Node.js) - Verify installation with `npm --version`

## Installation

1. Clone this repository (or navigate to this directory if you already have it)
2. Install dependencies:
   ```bash
   npm install
   ```

## Running Locally

To run the development server:

```bash
npm run dev
```

The app will be available at `http://localhost:5173` 

## Building for Production

To build the app for production:

```bash
npm run build
```

The built files will be in the `dist/` directory.
